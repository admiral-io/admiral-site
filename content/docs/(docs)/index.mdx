Admiral is a control plane for coordinating infrastructure and application delivery across environments.

> **Status:** These docs are in active development. Structure and examples are being expanded, and some sections are still rough drafts.

## What is Admiral?

Admiral provides a single pane of glass for managing deployments, infrastructure-as-code workflows, and environment configuration. It combines a server control plane with distributed execution components that run wherever your workloads live.
